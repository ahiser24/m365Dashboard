    Users by platform

    Users are counted for each platform they accessed (e.g., Web, Teams, mobile apps). A single user can appear in multiple categories if they used more than one access method, so totals will exceed the number of unique users.

    Web vs. mobile access

    Each user is counted once and categorized based on their activity across all devices. “Used mobile” includes anyone who accessed Viva Engage from a mobile or integrated app (e.g., Teams or mobile devices) at least once. “Web only” represents users who accessed Viva Engage exclusively through a browser
